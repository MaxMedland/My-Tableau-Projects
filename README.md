# 📊 Tableau Data Visualisation Project — Data Technician Bootcamp

This repository showcases my Tableau project completed during Week 2 of the **Data Technician Bootcamp**, where I explored global health, retail, and music‑streaming datasets to develop strong data‑visualisation and storytelling skills. The project focused on building interactive dashboards, applying filters and slicers, and designing clear, insightful visuals for decision‑making.

---

## 🚀 Project Overview

Using Tableau Public, I created a series of visualisations and dashboards based on the **Gapminder Health** dataset and an extended **Spotify Features** dataset. The goal was to uncover trends, compare metrics across countries and continents, and communicate insights through intuitive, interactive visuals.

The project emphasised:

- Turning raw data into meaningful insights  
- Designing dashboards for non‑technical audiences  
- Using visual analytics to support organisational decision‑making  

---

## 🧠 Key Skills Demonstrated

### 📈 Interactive Dashboards
- Built multi‑sheet dashboards combining bar charts, line charts, pie charts, scatter plots, and custom visuals  
- Applied interactive filters (Country, Continent, Year)  
- Ensured filters applied across relevant worksheets for a smooth user experience  
- Published dashboards to Tableau Public for sharing and collaboration  

---

### 🔍 Filters, Slicers & Interactivity
- Used **Top N filters** to isolate the top 5 countries by life expectancy  
- Added **single‑value dropdown filters** for gender and country selection  
- Enabled **continent‑level filtering** for scatter plots  
- Applied **global filters** across multiple sheets where appropriate  

---

### 🧮 Calculated Fields & Data Preparation
- Created calculated fields for:
  - Average metrics (Life Expectancy, BMI, Population)  
  - Percentage of total (e.g., gender population distribution)  
- Used table calculations to convert raw values into percentages  
- Cleaned and prepared datasets for visualisation  

---

### 📊 Chart Types Used

- **Bar charts** — Life Expectancy by Continent  
- **Line charts** — Life Expectancy trends over time  
- **Pie charts** — Gender population distribution  
- **Scatter plots** — BMI vs Life Expectancy  
- **Custom visuals** — Genre and key analysis for Spotify data  
- **Maps** — Geographic comparisons

## 📈 Insights Gained

- Life expectancy varies significantly by continent, with clear upward trends in many regions.  
- Gender population distributions reveal demographic imbalances that can inform healthcare planning.  
- BMI and life expectancy show continent‑level clustering, highlighting lifestyle and socioeconomic differences.  
- Spotify data shows that upbeat tracks tend to be more popular, and certain musical keys dominate modern genres.
